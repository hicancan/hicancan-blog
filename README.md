# hicancan-blog

A personal blog built with Astro, Tailwind CSS, and React. Focused on posts, projects, friends, and a guestbook.

## Requirements
- Node.js 18+ (Astro v5)
- npm

## Scripts
- `npm run dev` Start the dev server
- `npm run build` Build the site
- `npm run preview` Preview the build

## Project Structure
- `src/pages/` Routes and page content
- `src/components/` UI components
- `src/content/` Content collections (blog, friends, projects)
- `src/styles/` Global styles
- `public/` Static assets

## Deployment
Configured for GitHub Pages via `.github/workflows/deploy.yml`.
